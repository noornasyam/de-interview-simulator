This folder contains the curated AWS question bank for the DailyDEHub AI Interview Coach hybrid question system.

## Coverage

The bank focuses on AWS data engineering interview concepts rather than generic AWS trivia:

- S3 data lake zones and ingestion patterns
- IAM roles, least privilege, encryption, and auditability
- AWS Glue, Glue Data Catalog, crawlers, and serverless ETL
- Athena query patterns, partitioning, Parquet, and bytes scanned
- Redshift loading, performance, and workload placement
- Lambda, Step Functions, SQS/SNS, Kinesis, and orchestration trade-offs
- Lake Formation and governed data lake access
- CloudWatch monitoring, incident response, and production troubleshooting
- Cost optimization across S3, Athena, Glue, Redshift, EMR, and streaming workloads
- Reliability, DR, multi-account architecture, and enterprise governance

Beginner and Junior questions focus mostly on objective concepts. Mid-Level mixes objective and scenario-based questions. Senior, Lead, and Architect levels emphasize production design, troubleshooting, governance, cost, security, reliability, and enterprise architecture.
