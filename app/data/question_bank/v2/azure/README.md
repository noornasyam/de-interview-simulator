This folder contains the curated Azure question bank for the DailyDEHub AI Interview Coach hybrid question system.

## Coverage

The bank focuses on Azure data engineering interview concepts rather than generic Azure trivia:

- Azure Data Lake Storage Gen2 and Blob Storage concepts
- Azure Data Factory orchestration, linked services, triggers, and copy patterns
- Synapse Analytics and serving-layer choices
- Azure SQL / SQL Database basics in ingestion and serving scenarios
- Azure Databricks and Spark troubleshooting on Azure
- Event Hubs, Service Bus, Azure Functions, and Logic Apps basics
- Microsoft Purview catalog, classification, lineage, and governance
- Managed identities, Key Vault, RBAC, ACLs, and private endpoints
- Azure Monitor and Log Analytics for production operations
- Partitioning, Parquet/CSV/JSON, ingestion patterns, and medallion lake zones
- Cost optimization, reliability, DR, security, and enterprise architecture

Beginner and Junior questions focus mostly on objective concepts. Mid-Level mixes objective and scenario-based questions. Senior, Lead, and Architect levels emphasize production design, troubleshooting, governance, cost, security, reliability, and enterprise architecture.
