## Type Mapping

The shared v2 schema uses a fixed set of supported question types. Git-specific ideas such as workflow management, release strategy, operating model, enterprise Git strategy, platform engineering, and SDLC governance are represented with supported types such as `scenario`, `troubleshooting`, `architecture`, `reliability`, `governance`, `stakeholder_impact`, `platform_design`, `security`, and `enterprise_architecture`.
